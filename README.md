 🏥 Hospital Management Analytics Dashboard

 📌 Project Overview

This project analyzes hospital patient data using Excel, SQL, and Power BI. The objective is to identify trends in patient demographics, medical conditions, insurance types, treatment outcomes, and hospital stay duration through interactive visualizations.

---

 🛠️ Tools Used

- Microsoft Excel
- MySQL
- Power BI
- GitHub

---

 📂 Dataset

The dataset contains patient information, including:

- Patient ID
- Age
- Gender
- Medical Condition
- Admission Type
- Insurance Type
- Smoking Status
- Region
- Treatment Outcome
- Length of Stay
- Income

---

 📊 Dashboard Features

- Total Patients KPI
- Average Age KPI
- Average Income KPI                                                       [Hospital Dashboard](Images/Dashboard.png)
- Average Length of Stay KPI
- Patients by Medical Condition
- Gender Distribution
- Patients by Region
- Treatment Outcome Distribution
- Insurance Distribution
- Smoking Status Analysis
- Admission Type Analysis
- Interactive Slicers

---

 🗄️ SQL Analysis

The project includes SQL queries for:

- Total Patients
- Average Age
- Gender Distribution
- Medical Condition Analysis
- Insurance Analysis
- Region-wise Analysis
- Ranking Medical Conditions
- Age Group Analysis using CASE WHEN

---

 📈 Business Insights

- Identified the most common medical conditions.
- Compared patient distribution across regions.
- Analyzed insurance coverage among patients.
- Evaluated treatment outcomes.
- Calculated average hospital stay.
- Explored demographic trends using age and gender.
